Angular Context Engineering

An Angular 21 starter template optimized for Claude Code AI agents

Build features faster with the PRP (Product Requirement Prompt) workflow. This template provides Claude Code with the context needed to produce consistent, architecture-compliant code.

What is this repository?

This is not a standard Angular starter. It's a structured environment that gives Claude Code AI the context needed to produce consistent, architecture-compliant code.

The key innovation is the PRP (Product Requirement Prompt) workflow, which separates planning from implementation to reduce hallucinations and enforce coding standards.

  • Angular 21 with Signals, Standalone Components, and Modern Control Flow
  • Zoneless architecture with OnPush change detection
  • Tailwind CSS 4 for styling
  • Structured .ai/ directory with architecture documentation
  • Architect Agent for automated code review
